📋 Quick Setup Guide

  1. Start the server: Run npm start in the backend directory
  2. Generate QR codes: Visit the QR Generator page and create codes for your tables
  3. Print QR codes: Place them on restaurant tables
  4. Open dashboard: Keep the dashboard open on your counter/kitchen device
  5. Start taking orders: Customers scan QR codes to order!
